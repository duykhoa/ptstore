# PIVOTAL STORE

This repository makes some statistic on Pivotal Tracker accepted stories

## Install

1. Clone the project
1. install dependency (npm install & bower install)
1. Add config file, follow `.config.yaml.sample`
1. Using the command

  ```sh
    ruby -Ilib bin/repl.rb
  ```

1. Start node server by command `polymer serve node_server`
1. Visit the application
1. If it doesn't work, raise me an issue
